¿Cómo instalar VirtualBox 2025 en Windows 11?

Sigue estos pasos:

  1. Descarga el instalador desde el sitio oficial de Oracle.
  2. Ejecuta el archivo VirtualBox-7.0.x-xxxxxx-Win.exe como administrador.
  3. Acepta los permisos de instalación de dispositivos.
  4. Reinicia tu equipo después de la instalación.

Nota: Requiere habilitar Virtualización en la BIOS.

¿Cómo simular un entorno de dominio con Windows Server?

Pasos básicos:

  1. Crea una VM con Windows Server 2022.
  2. Instala el rol Active Directory Domain Services.
  3. Configura el DNS y las direcciones IP estáticas.
  4. Une máquinas virtuales con Windows 11 al dominio.

Ver guía detallada aquí (próximamente).

¿VirtualBox 2025 soporta Windows 11 con TPM 2.0?

Sí, pero necesitas habilitar manualmente el TPM virtual:

# Ejecutar en CMD (Host):
cd "C:\Program Files\Oracle\VirtualBox\"
VBoxManage modifyvm "NombreVM" --tpm-type=2.0 --firmware=efi

Luego instala Windows 11 normalmente.
